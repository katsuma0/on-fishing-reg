Notes

On selling. Copy and silhouettes are original; the ad lines are written for this project, not lifted from any manufacturer. Marque & model names are trademarks, so for a print-on-demand launch a "safe mode" swaps the marque for era + chassis code + origin (e.g. TYPE 993 · GERMANY · 1993) and keeps every other element identical. The reference site autocatalogarchive.com is blocked by this environment, so its PDFs weren't used or needed.
